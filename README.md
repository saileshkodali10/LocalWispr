# LocalWispr

Offline speech-to-text application inspired by Wispr Flow.

## Features

- Faster Whisper transcription
- Push-to-talk recording
- Auto paste
- Offline processing
- No API keys
- No cloud dependency

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
py -3.13 hotkey.py
```

## Hotkey

Ctrl + Win

Hold to record.

Release to transcribe.
